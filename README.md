# clone-kodak-paris

Maquette wireframe : https://wireframe.cc/Gd4jnx

* Police d'écriture :
    * "Open Sans", sans-serif;

* Couleurs utilisées :
     * grey: rgb(72,84,97)
     * black:rgb(3, 3, 3)
     * shadow:  rgba(0, 0, 0, 0.15)
     * yellow: #ffb700
     * yellow-opacity: rgba(255, 204, 0, 0.75)

    * icone chevron droite et gauche : `fas fa-chevron-left` , `fas fa-chevron-left`
